Reports and Background Tasks

Reports, Monte Carlo Analysis and Scheduled Task are accessible here

When a Report or Monte Carlo Analysis is submitted, it is processed on the server and the status is updated here. The status is automatically updated on this page by the server, but Refresh can be seleccted to get the latest status of your request.

When a Scheduled Report has been executed, the results will appear here.

  1. Use the hyperlinks in the table to access the report
     
  2. Delete old reports by selecting the reports and pressing Delete
     
  3. Use New to create a new scheduled report.
    If you select Email the report will be emailed to your registered email account as well as appearing in the list above
     
  4. Double click on existing Scheduled Reports to edit them
