Risk Explorer

  1. Select a project from the Navigator, above, the risks for the selected project are displayed in the Risk Explorer Table
     
  2. Optionally select Include Descendant Project to also display the risks from all sub projects.
     
  3. From the Action menu, select Show Filter to display the top filters panel or Show Risk Summary Filter to display the bottom detail preview panel.
     
  4. Double click on the risk in the Risk Table to open it in the editor.
     
  5. Single click, or shift-click to select multiple risks. When risks are selected, choose actions from the Action menu or right click to select actions from the context menu
     
  6. Risks can be created or deleted by selecting the appropriate item in the Action menu or the Risk Table context menu.
     
  7. Click on the column title in the Risk Table to sort or group by the selected column.
     
  8. To reassign risk to another project, click and drag the risk from the Risk Table to the target project in the Project Navigator.
     
  9. Use the Goto: selector to switch to other main screens.
     
  10. Reports
     
      Risk Reports are produced based on the currently selected set of risks.
     
      Registry Reports are produced based on the current state of the Risk Table, including filters and sortation criteria
     
      Project Reports are produced based on the current selected project, including sub projects if selected.
     
      Repository Reports are produced based on the current Repository.
     
  11. Right click on items in the Project Navigator to select project actions.
     
  12. Account Options can be set via the items in the Action menu.
