Risk Ranking

  1. Select and drag the risk to it's new ranking
     
  2. By default, risks are ranked according to their tolerance, probability and impact.
     
  3. Color of each item corresponds to the Current Tolerance of the risk .
     
  4. Use the Project Navigator to select a different project.
     
  5. Details of the risk are displayed when the mouse passes over each item.
     
  6. Double Click on any item to open the Risk Editor for the item.
     
  7. Select Save to save any ranking changes.
     
  8. Select Reset to undo any changes since the last save.
